# QUICK FIX REFERENCE - Undrop Error

## The Error
```
SQLSTATE[23000]: Integrity constraint violation: 1452 
Cannot add or update a child row: a foreign key constraint fails
FOREIGN KEY (`student_id`) REFERENCES `students` (`id`)
```

## What Was Wrong
System tried to insert undrop record for a student/teacher that doesn't exist.

## What's Fixed
✅ Added exception handling to BOTH undrop locations
✅ Catches foreign key constraint errors
✅ Shows friendly error messages
✅ Validates before and after insert
✅ Handles race conditions
✅ Database integrity verified

## Two Places Fixed

### 1. Admin Dashboard Undrop
**File**: `frontend/admin/dropped_cards.php` (Lines 61-104)
- When admin clicks "Undrop" on a single card
- Now wraps INSERT in try-catch
- Handles FK violations gracefully

### 2. Bulk Undrop API
