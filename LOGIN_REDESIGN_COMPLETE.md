## 🔧 Customization Guide

### Change Primary Color
```css
/* In CSS or HTML */
--primary-color: #YOUR_COLOR;
/* Automatically updates buttons, borders, glows */
```

### Adjust Card Size
```css
.login-wrapper {
    max-width: 450px; /* Change this value */
}
```

### Modify Animation Speed
```css
.shape-1 { animation: float 6s ease-in-out infinite; }
/* Change 6s to any duration */
```

### Change Background Gradient
```css
.login-page {
    background: linear-gradient(135deg, #COLOR1 0%, #COLOR2 50%, #COLOR3 100%);
}
```

### Disable Animations (for accessibility)
```css
@media (prefers-reduced-motion: reduce) {
    * { animation: none !important; }
}
```

## 📞 Support

### Common Issues

**Shapes not animating?**
- Check browser compatibility (use latest version)
- Verify CSS is loading (browser DevTools → Network)
- Ensure JavaScript enabled in browser

**Fonts look different?**
- Clear browser cache (Ctrl+Shift+Del)
- Check internet connection (for Google Fonts)
- Try different browser

**Mobile layout broken?**
- Check viewport meta tag in HTML
- Test with DevTools device emulation
- Verify media queries are loading
